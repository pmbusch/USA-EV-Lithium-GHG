# The energy, material and carbon handprint of lithium-ion batteries in electric vehicles - A US fleet model

Scripts to perform all the required spatial joins at county level to merge spatial data at different resolutions: electricity balancing systems, transport regions from EIA, population data, ecoinvent upstream electricity factors.

* **Census.R**: Joins Census data to other regions aggregators.
* **EnergyRegions.R**: Load EMM Shapefile for EIA electricity forecast.
